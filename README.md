# Chatbot from scratch with Next, AI SDK, OpenAI, and Shadcn/UI

## Getting Started

First, clone the project:

```shell
git clone https://github.com/makiessedev/chat-ai.git && cd chat-ai
```

Install dependencies:

```shell
npm install
```

Run project:

```bash
npm run dev
```

### att

You need to set env variable:

OPENAI_API_KEY=xxxx xxxx xxxx

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
